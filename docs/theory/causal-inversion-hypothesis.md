# The Causal Inversion Hypothesis

Source document: `causal_inversion.docx`.

## Thesis
The Causal Inversion Hypothesis (CIH) proposes that singularities are not physical terminations but causal horizon boundaries where the interpretation of volume and density inverts across a transition into a child-universe coordinate system.

## Core idea
Instead of treating `r = 0` as a literal infinite-density endpoint, CIH treats it as the pole of an inversion map:
- parent volume maps into child time
- parent density maps into child spatial compression / expansion coordinates

In the source draft this is written as a formal boundary transformation with:
- `tau_child ~ l^4 / V_parent`
- `chi_child ~ rho_0 / rho_parent`

## Why it matters to SoCT
CIH is the formal bridge between:
- black-hole collapse in a parent universe
- initial conditions in a child universe
- directional and memory transfer across the causal boundary

It is the geometric / causal extension that allows SoCT memory to persist into cosmology.

## Claimed explanatory reach
The source draft uses CIH to address:
- the meaning of singularities
- the black-hole information problem
- cosmological initial conditions
- parent-to-child memory transfer
