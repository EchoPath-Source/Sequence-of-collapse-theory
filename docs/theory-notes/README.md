# Theory Notes

This folder holds speculative and conceptual extensions to the Sequence of Collapse Theory.

These notes are **not** peer-reviewed claims and should not be treated as validated physics. They are research scaffolds: places to preserve hypotheses, clarify terminology, identify what would have to be true, and separate conceptual exploration from reproducible experiments, datasets, and manuscript-grade papers.
