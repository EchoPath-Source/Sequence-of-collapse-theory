# Recursive-Universe Status Note

**Evidence label:** `exploratory-simulation`

## Status
This branch of work remains exploratory.

## Preserved purpose
The recursive-universe line was used to explore whether parent/child-universe transfer ideas, transfer kernels, and toy CMB-like outputs could be made:
- internally coherent
- observationally constrained in principle
- explicitly falsifiable in at least some parameter ranges

## Preserved limitation
The exploratory branch does **not** currently count as a validated SoCT cosmology result. It should remain separated from:
- OSF prereg claims
- validated PM simulation outputs
- publication-facing evidence summaries

## Why preserve it anyway
Because it may still be useful later for:
- hypothesis generation
- identifying missing constraints
- understanding where speculative branches fail
- planning future toy-model-to-rigorous-model transitions
