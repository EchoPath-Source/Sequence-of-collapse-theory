# The Engramon Scale

Source document: `engramon_scale.docx`.

## Thesis
This note tries to pin the CIH inversion scale `l` by tying Kerr-ring geometry to the observed CMB infrared cutoff / low-quadrupole suppression.

## Main result preserved from the source
The draft concludes that the inversion scale is not simply the Planck length. Instead it points toward a derived intermediate scale, named the **Engramon scale**.

## Why the note is important
Unlike some more speculative theory notes, this one explicitly preserves an open problem instead of claiming closure.

The central unresolved issue is the scale bridge:
- parent Kerr ring circumference is microscopic / astrophysical relative to cosmological scales
- observed CMB cutoff is macroscopic
- a conformal amplification factor is required, but its correct form is not yet fixed by the theory
