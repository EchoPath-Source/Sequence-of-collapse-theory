# OSF Materials

Use this folder for preregistration drafts, upload-ready PDFs, methods summaries, and versioned amendment notes tied to external registration.

## Recommended contents
- preregistration markdown drafts
- upload-ready PDF copies
- correction addenda
- version log of what was submitted and when
- links to public OSF components once published

## Rule
Every OSF-facing file should clearly state:
- document type
- date
- version
- whether it was uploaded externally
