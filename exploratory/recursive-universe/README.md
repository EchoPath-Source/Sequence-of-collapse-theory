# Recursive-Universe Exploratory Sandbox

**Evidence label:** `exploratory-simulation`

This folder is reserved for parent/child-universe, recursive-universe, transfer-function, and toy-CMB style explorations.

## Critical caution
Nothing in this folder should be treated as established Sequence of Collapse evidence unless it is independently upgraded through later derivation, validation, and reproducible testing.

## What belongs here
- toy-model analyses
- transfer-function sketches
- parent/child-universe exploratory notes
- falsifiability scans for speculative extensions
- illustrative figures that are useful for thinking but not yet paper-grade evidence

## Why keep it
These materials still matter because they:
- preserve idea lineage
- capture speculative branches worth revisiting later
- help prevent repeated work
- separate exploratory imagination from validated simulation lines
