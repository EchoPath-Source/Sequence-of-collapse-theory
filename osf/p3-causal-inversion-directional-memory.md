# P3 — Causal Inversion Hypothesis & Directional Memory Cosmology

## Overview

This project is the P3 directional-memory cosmology hub for the Sequence of Collapse Theory (SoCT). It investigates whether parent-universe, black-hole-bounce, or Kerr-to-Cosmos geometries can leave a directional memory imprint that survives into a child universe as large-scale cosmological anisotropy.

The project combines the Causal Inversion Hypothesis (CIH), Kerr-to-Cosmos geometry, and SoCT memory-field ideas.

The central working question is:

**Can a universe inherit directional memory from a prior causal domain, and can that memory be tested through CMB-scale anisotropy or preferred-axis signatures?**

## Core hypothesis

The working claim is that a universe born through a bounce, inversion, or black-hole-like transition may inherit directional memory from the parent geometry. This inherited memory may appear as statistically testable large-scale anisotropy.

Candidate signatures include:

- low-l multipole alignment;
- hemispherical power asymmetry;
- parity asymmetry;
- temperature-polarization correlation anomalies;
- E-mode / B-mode orientation signatures;
- large-scale structure alignment with a predicted memory axis.

## Theoretical branches

### 1. Causal Inversion Hypothesis

CIH proposes that causal order can invert or reorient across a high-curvature transition, such that what appears as an initial condition in the child universe may encode memory of a prior or parent state.

### 2. Directional memory

Directional memory is a residual axis, vector, or anisotropic imprint that persists after a bounce or transition.

### 3. Kerr-to-Cosmos geometry

This branch explores whether rotating black-hole or ring-like geometry provides a bridge between black-hole interiors, bounce cosmology, and universe-scale orientation.

### 4. Parent-universe memory

Parent-universe memory refers to inherited structure from a prior causal domain, treated in SoCT as a memory-field imprint that survives transformation across a boundary.
