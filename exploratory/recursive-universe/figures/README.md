# Recursive-Universe Figures

**Evidence label:** `exploratory-simulation`

This folder is reserved for figures generated by recursive-universe and parent/child transfer toy models.

## Current status
Figure landing structure is in place. Final figure files still need to be added or verified.

## What belongs here
- transfer-function visualizations,
- child-CMB toy maps,
- angular-power toy plots,
- non-Gaussianity / falsifiability plots,
- parent-child alignment diagnostic figures.

## Required metadata
Every figure should have a caption or source note recording:

1. generating code,
2. input parameters,
3. random seed,
4. output claim level,
5. whether the plot is illustrative, diagnostic, or part of a preregistered test.

## Rule
Do not cite these figures as real-universe evidence unless a later validated analysis upgrades the branch.
