# P1 — Memory-Field Gravity Phenomenology / SPARC / SoCT-PM

## Overview

This project is the P1 empirical-gravity hub for the Sequence of Collapse Theory (SoCT). It investigates whether gravity can be modeled not only as curvature sourced by present mass-energy, but also as an accumulated memory effect of prior collapse events.

The central working question is:

**Can gravitational behavior at galactic scale be partly explained by accumulated collapse history stored as a persistent memory field?**

This project connects theoretical work on Gravity as Accumulated Collapse History with SoCT-PM simulations and SPARC rotation-curve validation.

## Core hypothesis

The Sequence of Collapse framework proposes that collapse events leave persistent spacetime-orientation imprints. Across many events and long timescales, these imprints accumulate into an effective memory field. This memory field may behave gravitationally or modify apparent gravitational dynamics.

In this model:

1. Matter and light interactions generate local collapse events.
2. Collapse events leave memory-like orientation imprints.
3. Repeated collapse history accumulates into a field-like residue.
4. At galactic scales, this residue may contribute to rotation-curve behavior.

## Empirical anchor: SPARC

The SPARC dataset provides high-quality disk-galaxy rotation curves and baryonic mass models. P1 uses SPARC as a first empirical test bed because it allows comparison between observed acceleration, baryonic distribution, and residual gravitational discrepancy patterns.

## Research aims

- Test whether memory-field terms improve fits to galaxy rotation curves.
- Compare memory-field behavior with dark-matter halo models and MOND-like relations.
- Search for correlations between inferred memory terms and galaxy age, morphology, star-formation history, or cumulative collapse proxies.
- Develop SoCT-PM simulations that implement collapse-memory dynamics.
- Define falsification conditions and negative-result criteria before interpreting positive results.
