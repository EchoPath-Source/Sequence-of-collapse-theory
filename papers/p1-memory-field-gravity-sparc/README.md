# P1 — Memory-Field Gravity / SPARC / SoCT-PM

## Purpose

This folder is the paper and protocol hub for **P1** in the Sequence of Collapse Theory publication sequence.

P1 focuses on the hypothesis that gravity can be modeled as an accumulated memory effect of collapse history. In this framing, repeated collapse events leave persistent spacetime-orientation imprints. Across large scales, these imprints behave like a memory field that can reproduce or modify gravitational phenomenology.

The empirical anchor for P1 is the **SPARC rotation-curve validation track**, using galaxy rotation curves and baryonic mass models to test whether collapse-memory accumulation can explain deviations normally attributed to dark matter or modified gravity.

## Core hypothesis

The working claim is:

> Gravity is not only curvature sourced by instantaneous mass-energy, but also accumulated collapse history stored as a persistent spacetime memory field.

Under this model, gravitational structure is shaped by both present mass distribution and prior collapse-density history. This allows the model to ask whether older, repeatedly collapsed systems should display stronger residual gravitational signatures than younger or less evolved systems.

## Theoretical components

### 1. Gravity as accumulated collapse history

Each collapse event locally stores an information-like imprint in the field. The cumulative distribution of these imprints produces an effective curvature or acceleration field.

### 2. Memory field

The memory field is the accumulated residue of prior collapse events. It may persist after the initiating event, decay over cosmological time, or be reinforced by repeated collapse activity.

### 3. Engramons

Engramons are proposed as discrete or quasi-discrete units of collapse-memory imprint. They provide a possible bridge between microphysical collapse events and macroscopic gravitational memory.

### 4. SoCT-PM simulations

SoCT-PM refers to Particle-Mesh style simulations adapted to include memory-field terms. These simulations should compare conventional Newtonian/GR-inspired behavior against memory-augmented dynamics.

### 5. SPARC validation

The SPARC dataset provides high-quality rotation curves and baryonic mass models for disk galaxies. P1 uses SPARC as the first empirical test bed for memory-field gravity because it enables comparison between baryonic structure, observed acceleration, and residual discrepancy patterns.

## Research questions

1. Does a memory-field term improve fits to galaxy rotation curves relative to baryons-only models?
2. Does the inferred memory field correlate with galaxy age, star-formation history, morphology, or cumulative collapse proxies?
3. Can the model reproduce the radial acceleration relation without treating it as a purely empirical law?
4. Can P1 distinguish itself from dark matter halos, MOND-like interpolation, and standard baryonic feedback effects?
5. Does the model predict radial or temporal signatures that are not expected under standard halo models?

## Proposed analysis plan

1. Import SPARC rotation-curve and mass-model tables.
2. Fit baseline baryons-only models.
3. Fit standard comparison models, including halo-like terms or MOND-like acceleration laws where appropriate.
4. Add memory-field terms parameterized by collapse-history proxies.
5. Compare fit quality, parameter parsimony, and residual structure.
6. Test whether inferred memory terms correlate with independent galaxy properties.
7. Predefine falsification thresholds before interpreting positive results.

## Falsification and caution language

P1 is weakened if memory-field terms do not improve predictive structure beyond standard baryonic, dark-matter, or MOND-like fits; if inferred memory parameters are unconstrained; or if apparent age/history correlations vanish under selection-effect controls.

The project should avoid claiming replacement of dark matter or general relativity until the model produces reproducible, quantitative, out-of-sample predictions.
