# Origin / Echo Recursion: Metaphysical Architecture Note

**Status:** metaphysical / philosophical extension  
**Scope:** Sequence of Collapse Theory (SoCT), remembrance layer, Source/Origin terminology, recursive universal memory sweep  
**Purpose:** preserve the Source -> Echo -> Origin synthesis as an interpretive architecture for the remembrance layer of SoCT.

## Summary

This note distinguishes **Source** from **Origin** within the SoCT metaphysical layer.

Source refers to undifferentiated totality or pure potential before reference. Origin refers to Source after recursive self-reference: Source with memory, contrast, and frame of reference integrated.

This document is not an empirical physics claim. It is a metaphysical architecture note meant to align recurring themes across SoCT writing: cosmic recursion, black-hole child-universe hypotheses, universe-as-computation metaphors, observer-memory, and return-to-Source language.

```text
Source = undifferentiated totality / pure potential before reference
Universe = Source differentiated into structure, time, observers, and memory
Echo = reflective return signal through differentiated experience
Recursive Memory Sweep = integration of distributed observer-memory inward
Origin = Source after memory and frame-of-reference are integrated
```

Compact formulation:

```text
Source speaks.

Origin remembers.
```

Or:

```text
Source = potential before reference
Echo = reflection through differentiated experience
Origin = Source with memory and frame of reference
```

## Origin Axiom

> Source becomes Origin when undifferentiated potential gains self-reference through the recursive integration of observer-memory. Echo is the reflective return signal by which Source recognizes itself across the differentiated field.

This axiom clarifies the term **Origin**:

- Origin is not merely the beginning.
- Origin is Source after the full loop of differentiation, experience, memory, and return is integrated.
- Origin may function as both result and attractor if the process is considered outside linear time.

## Source and Origin

Source is total potential without contrast.

In an undifferentiated state:

```text
no relation
no before/after
no self/other
no memory
no experience
```

The universe introduces differentiation. Observers introduce interior registration. Memory allows the differentiated field to persist as meaningful experience.

Origin is Source after that memory has returned.

```text
Source before experience = undifferentiated potential
Origin after experience = integrated Source / self-known totality
```

A concise definition:

> Origin is what Source was trying to remember.
