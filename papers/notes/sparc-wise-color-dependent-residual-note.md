# SPARC / WISE color-dependent residual note

This repository note records the status of the short write-up built from the SPARC rotation-curve decomposition analysis.

## Working paper claim
Using public SPARC rotation-curve mass-model decompositions, define an apparent dark residual:

`fDM(r) = 1 - Vbar^2(r) / Vobs^2(r)`

with

`Vbar^2 = Vgas^2 + Vdisk^2 + Vbulge^2`

Then test the residual against WISE stellar-mass data and `g-W1` color.

## Main result currently preserved
The residual structure correlates with the WISE color proxy and the relationship strengthens when restricting to:
- SPARC `Q = 1` high-quality curves
- disk-dominated systems

The result remains visible at multiple residual definitions and standardized radii.
