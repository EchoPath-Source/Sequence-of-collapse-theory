# P1 Result Tables

This folder stores machine-readable result tables for the canonical P1 SPARC / age-dependent rotation-curve package.

## Current regenerated text/CSV outputs

Regenerate from the repository root with:

```bash
python papers/p1-age-dependent-rotation-curves-sparc/analysis/sparc_age_fdm_analysis.py
```

| File | Status | Notes |
|---|---|---|
| `SPARC_age_fdm_correlations.csv` | regenerated | Pearson and Spearman correlation between `age_best` and `fdm_outer_mean`; N=175. |
| `SPARC_age_fdm_partial_correlations.csv` | regenerated | Partial-correlation checks controlling for `logVmax`, `logRmax`, and `logSB0`; N=175. Full structural-control row `Vmax_Rmax_SB0` gives `partial_r ≈ 0.107`, `p ≈ 0.159`. |
| `SPARC_age_fdm_regression_models.csv` | regenerated | OLS age-coefficient table across control models. Full structural-control model `fdm ~ age_best + logVmax + logRmax + logSB0` attenuates the age coefficient to `p_age ≈ 0.162`. |
| `SPARC_age_fdm_binned_bootstrap.csv` | regenerated | Mass-bin bootstrap results for lowV, midV, and highV bins. |

## Claim boundary

These tables are result artifacts, not a complete reproducibility package by themselves. They should be paired with:

- the source/derived SPARC dataset;
- the analysis script or notebook used to generate the tables;
- data provenance notes;
- controls/confounds documentation;
- independent reproduction before publication-grade claims.

Use:

> These result tables report candidate SPARC age/fDM correlations and controls pending full independent reproduction and audit.

Avoid:

> These tables prove SoCT, prove memory gravity, or prove that dark matter is collapse memory.
