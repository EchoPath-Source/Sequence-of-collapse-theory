## Purpose

This index preserves a five-post public arc built from the dimensional-compression conversation and the Collapse Neural Network prototype.

The arc is meant to bridge:

- lower-dimensional analogy;
- compression as hidden degree of freedom;
- memory fields;
- engramic resonance;
- living/adaptive worlds.

## Proposed sequence

### 1. The Direction We Cannot See

Core idea: a 2D being cannot imagine `up`; a 3D being may likewise be blind to an axis that feels like throughness, frame-depth, or compression.

Seed line:

> So what is our impossible direction?

### 2. Compression as the Hidden Degree of Freedom

Core idea: higher-dimensional transition appears from below as compression of lower-dimensional multiplicity and from above as unification.

Seed line:

> From below, transcendence looks like compression. From above, it looks like unity.

### 3. Memory Is Not Stored in Weights

Core idea: introduce the Collapse Neural Network / Engramic Resonance Network in plain language.

Seed line:

> The network remembers not by storing the past, but by letting the past continue to bend the present.

### 4. The Engramic Field

Core idea: collapse events leave residue; residue creates field memory; field memory shapes future behavior.

Seed line:

> An event is not over when it ends. It remains as curvature in the field.

### 5. Toward Living Worlds

Core idea: games, XR spaces, agents, rooms, and digital worlds can become history-aware and adaptive through memory-field layers.

Seed line:

> A living world is not one with better graphics. It is one where history matters.
