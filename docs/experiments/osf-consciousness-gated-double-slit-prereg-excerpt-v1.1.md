## Experiment
Consciousness-gated double-slit / which-path test.

## Purpose
Test whether conscious access to quantum measurement outcomes is required for full wave-function collapse, under adversarial controls designed to separate:
- measurement
- recording
- deletion / inaccessibility
- conscious viewing

## Core conditions preserved from the working prereg package
- conscious access control
- recorded but unobserved condition
- measured then deleted condition
- sham-detection negative control

## Notes preserved from the source
- pre-registration first, then results preprint, then final peer-reviewed version
- publish regardless of positive or null result
- avoid sensationalism in media framing
- explicitly value rigorous null results

## Risk controls preserved
- calibration and backup instrumentation
- third-party blinding / key control
- pre-specified primary tests
- locked analysis code
- no optional stopping
- adversarial analyst / skeptical collaboration
