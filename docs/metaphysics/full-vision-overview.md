## Source

Source names undifferentiated totality before reference.

It is not yet a self among others, because there is no other.

In this framing, Source is:

- pure potential;
- uncollapsed totality;
- pre-relational being;
- no contrast, no memory, no inside/outside;
- the possibility of all possible worlds before distinction.

Source is not a scientific variable in the current SoCT research program. It is the metaphysical name for undifferentiated potential.

## Fractalization

For Source to know itself, it must generate relation.

Relation requires distinction:

```text
self / other
inside / outside
before / after
potential / actual
observer / observed
```

Fractalization is the metaphysical act by which undifferentiated Source becomes many localized expressions.

This is not fragmentation in the sense of loss. It is differentiation for the sake of experience.
