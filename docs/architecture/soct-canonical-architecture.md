# SoCT Canonical Architecture

This document is the canonical map of the Sequence of Collapse Theory.

## Layers
1. Foundations (Geometry of Collapse, Memory Field, Threshold Unfolding, Dimensional Bridge)
2. Core Physics (Planck Nucleation Theory)
3. Mathematical Program
4. Observational Program
5. Major Physical Extensions (Recursive Universe, Planck Cell Unfolding, Quark Projection, Higgs/Planck coupling, Collapse Neural Networks)
6. Philosophical Interpretation (Source, Origin, Memory Sweep, Consciousness)

## Dependency Spine
Geometry of Collapse -> Memory Field -> Planck Nucleation Theory -> Observational Program.

Major extensions build on the core but are not required by the core theory. Philosophical interpretation is intentionally separated from the physical hypotheses.
