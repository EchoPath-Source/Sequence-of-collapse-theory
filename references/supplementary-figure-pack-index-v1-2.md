## Purpose
This index records why the supplementary figure pack matters and how it should be used in the Sequence of Collapse repository.

## Expected role of the figure pack
The figure pack appears to serve as a visual companion to simulation and theory notes. It is best treated as:
- a figure-reference index
- a visual support layer for simulations
- a later source for manuscript-ready or preregistration-ready figure exports

## Recommended use rules
1. Do not treat figures as evidence without linked source data or scripts.
2. When a figure is reused, connect it to:
   - originating document
   - evidence label
   - script or data source if available
3. Distinguish clearly between:
   - conceptual diagrams
   - synthetic demonstrations
   - real-data visualizations
