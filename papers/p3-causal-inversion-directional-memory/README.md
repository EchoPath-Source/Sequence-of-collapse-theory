# P3 — Causal Inversion Hypothesis & Directional Memory Cosmology

## Purpose

This folder is the paper and protocol hub for **P3** in the Sequence of Collapse Theory publication sequence.

P3 investigates whether parent-universe, black-hole-bounce, or Kerr-to-Cosmos geometries can leave a directional memory imprint that survives into a child universe as large-scale cosmological anisotropy.

The project combines the **Causal Inversion Hypothesis (CIH)**, **Kerr-to-Cosmos geometry**, and Sequence of Collapse memory-field ideas to ask whether cosmic microwave background anomalies, preferred-axis signatures, parity asymmetries, or hemispherical power imbalances can be interpreted as residual directional memory.

## Core hypothesis

The working claim is:

> A universe born through a bounce, inversion, or black-hole-like transition may inherit directional memory from the parent geometry, and that memory may appear as statistically testable large-scale anisotropy.

This does not assume that existing anomalies are necessarily real physical signals. Instead, P3 asks whether SoCT can generate quantitative predictions that are distinguishable from cosmic variance, systematics, foreground contamination, or kinematic effects.

## Theoretical components

### 1. Causal Inversion Hypothesis

CIH proposes that causal order can invert or reorient across a high-curvature transition, such that what appears as an initial condition in the child universe may encode memory of a prior or parent state.

### 2. Directional memory

Directional memory is a residual axis, vector, or anisotropic imprint that persists after a bounce or transition. It may appear as low-l multipole alignment, hemispherical asymmetry, parity asymmetry, or polarization structure.

### 3. Kerr-to-Cosmos geometry

The Kerr-to-Cosmos branch explores whether rotating black-hole or ring-like geometry provides a bridge between black-hole interiors, bounce cosmology, and universe-scale orientation.

### 4. Parent-universe memory

Parent-universe memory refers to any inherited structure from a prior causal domain. In SoCT language, this is treated as a memory-field imprint that survives transformation across the boundary.

## Research questions

1. Can CIH produce a preferred axis direction rather than a post-hoc fit to known anomalies?
2. Can the model predict amplitude ranges that are outside cosmic-variance expectation but consistent with observational constraints?
3. Does Kerr-to-Cosmos geometry provide a natural mathematical source for anisotropic memory?
4. Can P3 be distinguished from foregrounds, dipole leakage, survey mask effects, and other systematics?
5. Can P3 connect to P2 and P1 through a common collapse-memory formalism?

## Falsification and caution language

P3 is weakened if no quantitative axis or amplitude can be derived before comparison with data; if predicted axes shift freely across datasets; if amplitudes fall inside ordinary cosmic variance; or if the relevant anomalies are better explained by systematics, foregrounds, or kinematic effects.

The project should avoid treating known CMB anomalies as confirmation unless the model generates independent, reproducible, and pre-specified predictions.
