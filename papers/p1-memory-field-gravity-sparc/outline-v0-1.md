# P1 Draft Outline v0.1 — Memory-Field Gravity / SPARC / SoCT-PM

## Working title

**Gravity as Accumulated Collapse History: A Memory-Field Phenomenology Tested Against SPARC Rotation Curves**

## Abstract placeholder

This paper develops a physics-facing version of the Sequence of Collapse Theory claim that gravitational behavior may include an accumulated collapse-memory component. Rather than treating gravity only as a field sourced by instantaneous mass-energy, the model introduces a memory field generated by repeated collapse events and persistent spacetime-orientation imprints. The paper proposes a minimal phenomenological term for memory-modulated gravity and outlines a validation strategy using SPARC galaxy rotation curves, baryonic mass models, and comparison baselines including dark-matter halos and MOND-like acceleration laws.

## 1. Introduction

- State the problem: galaxy rotation curves show systematic deviations from baryons-only Newtonian expectations.
- Define the standard comparison space: dark matter halos, modified gravity, baryonic feedback, empirical radial acceleration relation.
- Introduce the SoCT hypothesis in cautious language: collapse history may leave persistent memory-like imprints that contribute to effective gravitational behavior.
- Clarify that this paper proposes a phenomenological research program, not a confirmed replacement for dark matter or GR.

## 2. Background and motivation

### 2.1 Standard gravitational interpretations

- Newtonian baryons-only expectation.
- Dark matter halo modeling.
- MOND-like acceleration relation.
- General relativity as the high-confidence large-scale framework.

### 2.2 Why SPARC is a useful first test bed

- High-quality rotation curves.
- Baryonic mass models.

## 3. Theory: memory-field gravity

### 3.1 Collapse events as memory sources

- Define collapse event density C(x,t).
- Treat each collapse event as contributing an imprint to a persistent memory field M(x,t).
- Memory can decay, saturate, or reinforce through repeated events.

### 3.2 Minimal phenomenological memory kernel

Suggested starting point:

```math
M(x,t) = \int K(|x-x'|, t-t') C(x',t') d^3x' dt'
```

where K is a spatial-temporal memory kernel.

### 3.3 Effective gravitational coupling

Working internal model:

```math
G_eff(x,t) = G_0 [1 + \alpha M(x,t)]
```

A minimal temporal saturation form:

```math
M(t) = 1 - e^{-t/\tau}
```

### 3.4 Physical interpretations

- Memory field as accumulated spacetime-orientation residue.
- Engramons as possible discrete units of collapse-memory imprint.
- SoCT-PM simulations as particle-mesh approximations to memory-field dynamics.

## 4. Predictions

### 4.1 Rotation-curve residual prediction

Older or repeatedly collapsed systems should show stronger outer residual acceleration if memory accumulation is real.

### 4.2 Radial structure prediction

Memory terms may be smoother and more spatially extended than instantaneous baryonic mass distribution.

### 4.3 Saturation prediction

The memory contribution should saturate or asymptote rather than grow without bound.

### 4.4 Environmental prediction

Dense, collapse-rich environments may differ from low-collapse void environments in effective gravitational or expansion behavior.

## 6. SoCT-PM simulation path

### 6.1 Simulation goal

Build a particle-mesh model where mass distribution creates collapse events, collapse events update M(x,t), and M(x,t) modifies effective acceleration.

### 6.2 Minimal simulation loop

1. Initialize particle distribution.
2. Estimate collapse-event density proxy.
3. Update memory field using kernel K.
4. Compute gravitational acceleration with memory correction.
5. Evolve particles.
6. Compare emergent rotation curves against baselines.

### 6.3 Validation targets

- Recover Newtonian behavior when alpha = 0.
- Avoid runaway memory accumulation.
- Reproduce qualitative flat rotation behavior only under constrained parameter regimes.
- Compare to SPARC-derived empirical patterns.

## 7. Falsification criteria

P1 is weakened if:

- memory-field terms do not improve predictive structure beyond established baselines;
- memory parameters are unconstrained or degenerate with standard halo terms;
- age/history correlations vanish under controls;
- the model requires galaxy-by-galaxy free tuning;
- out-of-sample predictions fail.

## 8. Limitations

- SPARC is not sufficient by itself to validate new gravity.
- Galaxy ages and collapse-history proxies are indirect.
- Dark matter and baryonic feedback remain strong competing explanations.
- Memory-field language must be translated into measurable quantities.
