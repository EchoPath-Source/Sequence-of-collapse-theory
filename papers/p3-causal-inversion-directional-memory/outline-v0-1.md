# P3 Draft Outline v0.1 — Causal Inversion / Directional Memory Cosmology

## Working title

**Causal Inversion and Directional Memory: Testing Parent-Universe Residue in Large-Scale Cosmological Anisotropy**

## Abstract placeholder

This paper develops the Causal Inversion Hypothesis (CIH) as a physics-facing branch of the Sequence of Collapse Theory. It asks whether a universe born through a bounce, inversion, or black-hole-like transition could inherit directional memory from a parent geometry. The paper translates this idea into a testable cosmological program centered on preferred-axis predictions, CMB low-l anomalies, parity asymmetry, hemispherical power imbalance, and possible links to Kerr-to-Cosmos geometry. The goal is not to treat existing anomalies as confirmation, but to derive pre-specified predictions that can be compared against public cosmological datasets and systematic explanations.

## 3. Theory: Causal Inversion Hypothesis

### 3.1 Causal inversion

CIH proposes that causal order can reorient across a high-curvature transition. What appears as an initial condition in the child universe may encode memory of a prior or parent domain.

### 3.2 Directional memory

Directional memory is a residual vector, axis, or anisotropic imprint that survives a transition and biases large-scale structure or radiation statistics.

### 3.3 Kerr-to-Cosmos bridge

A rotating parent geometry may provide a natural axis. The challenge is to derive whether any axis-like information can survive isotropization and appear in observable quantities.

### 3.4 SoCT memory-field interpretation

In SoCT language, the inherited axis is treated as a memory-field imprint rather than an arbitrary initial condition.

## 4. Minimal model

### 4.1 Axis parameterization

Define a preferred direction:

```math
\hat{n}_M = (l_M, b_M)
```

where l_M and b_M are galactic longitude and latitude.

### 4.2 Amplitude parameterization

Define a dimensionless memory amplitude:

```math
A_M \ge 0
```

### 4.3 Observational imprint

Possible first-order imprint:

```math
\Delta T(\hat{n}) = \Delta T_{\Lambda CDM}(\hat{n}) + A_M F(\hat{n} \cdot \hat{n}_M)
```

where F is a chosen anisotropy template to be defined before fitting.

### 4.4 Requirements for publication-quality model

- Axis must be predicted or constrained before data comparison.
- Amplitude must be bounded by theory or independent constraints.
- Template must not be freely tuned to known anomalies.

## 6. Methods

### 6.1 Literature-first phase

- Compile anomaly claims and counterclaims.
- Identify systematics and robustness tests.
- Separate robust observables from contested signals.

### 6.2 Model-first phase

- Derive axis and amplitude constraints before fitting.
- Define one or two templates only.
- Register null expectations and decision thresholds.

### 6.3 Data comparison phase

- Use public Planck and WMAP maps or published summary statistics.
- Compare template alignment against random isotropic simulations.
- Test whether axis is stable across masks, frequency bands, and estimators.

## 7. Falsification criteria

P3 is weakened if:

- no quantitative axis or amplitude can be derived before looking at data;
- the preferred axis shifts freely across datasets or analysis choices;
- amplitudes fall within ordinary cosmic variance;
- anomalies are better explained by foregrounds, survey masks, dipole leakage, or other systematics;
- the model cannot distinguish itself from post-hoc pattern matching.

## 8. Limitations

- Low-l anomalies are statistically fragile.
- Cosmic variance is large.
- Systematics and foreground modeling are difficult.
- A directional-memory model must make pre-specified predictions to be scientifically useful.
