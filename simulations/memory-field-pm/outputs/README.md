# Memory-Field PM Outputs

**Evidence label:** depends on generating code version.

This folder is reserved for CSV, JSON, and other machine-readable outputs generated by the memory-field particle-mesh simulation line.

## Current status
Structure is now in place, but production outputs still need to be added or verified.

## Required metadata for each output
Every output file added here should state:

1. code version used,
2. parameter set,
3. random seed or seed range,
4. baseline vs memory-active run,
5. whether the file is validation, exploratory, or production-sweep output.

## Recommended filenames

```text
v061_baseline_seed42_snapshots.csv
v061_memory_seed42_snapshots.csv
v061_parameter_sweep_summary.csv
v061_dwarf_survival_by_environment.csv
```

## Do not add
- screenshots without source data,
- unlabeled synthetic outputs,
- outputs from deprecated code unless clearly archived as historical provenance.
